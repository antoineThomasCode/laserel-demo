# Laserel Demo

Proposition commerciale interactive Laserel × Cohorte avec chatbot IA intégré.

## Stack
- Frontend: HTML/CSS/JS vanilla
- Backend: Node.js Express
- IA: Claude Sonnet 4.5 (Anthropic)
- Deploy: Docker + Traefik

## URL
`laserel.cohorte.fr`

## Dev local
```bash
cp .env.example .env
# Ajouter ANTHROPIC_API_KEY
npm install
npm run dev
```
